# 為什麼有這個 Blog？

資工系的學生擁有一個自己的 blog 似乎是一件很正常的事情，大一時聽到一些學長們說很推薦自己架一個 blog，拿來寫一些技術文章、寫一些關於自己的事情，從那時候開始就想自己架一個紀錄自己的 blog，加上我又很喜歡用文字記錄一堆有的沒的東西，像是每學期修課的心得、每一年的回顧等等，有一個地方紀錄的確是很重要的事情，於是誕生了架一個 blog 的想法

# 這個 Blog 有什麼？

在這邊你應該可以找到技術文章，基本上我會從一些基本的 tutorial 開始，某些我學過、用過的工具，寫一份盡可能完整的 tutorial，希望能幫助看到這份 tutorial 的人！除了 tutorial，當然還會有其他主題，沒意外會努力做一些資安相關的主題，CTF 的 writeup 應該也會有，如果之後比賽有厲害到可以寫 writeup 的話

總之，技術文章多數會跟資工相關，不過有機會也會偷偷放一些其他領域的，可以期待一下！

除了技術文章，其他部分應該會有更關於我的事情，包含做專案經歷、每年的回顧心得、出去玩的遊記等等

# 這個 Blog 的歷史

第一次架的 blog 是在大一暑假 CSI: Eden 資工營時 Jacob 的課程中嘗試，那時候使用到 Vue3, TailwindCSS，不過大一涉世未深又擁有潔癖的我就很希望所有東西都靠自己刻一個出來，雖然 Jacob 的那一份超級漂亮，但還是希望能有一個完全按照自己意志生出來的網站，所以在那年暑假和隔年寒假就開始花一些些時間學網頁三巨頭 HTML, CSS, JS

沒錯，當初無知的我就是想要靠著這三巨頭把自己的網站架出來，不得不說還真的有一些成果，不過很快就意識到現在的網頁不可能靠這幾種語言架出來，應該使用一些前端框架，所以第二個 blog 也就此報銷。記得之後有一天晚上在系館 B1 趕程設專題時跟 Corgi 聊天也剛好聊到這個問題，他推薦了 React，算是種下這次 blog 的種子

說要架 blog，明明不是多困難的東西，但被我架了這麼久還沒有一個成果，直到今年寒假（2024）才開始意識到問題的嚴重性，覺得自己真的要趕快生出一個可以建立自己品牌的地方，剛好現在能力終於有那麼一點點長進，便強迫自己今年寒假一定要生出來

現在看到的 blog 是採用 hexo 這個靜態網站生成器搭配 Shoka 主題所製成，詳細步驟可以參考這篇（暫定是下一篇更新的文章，再等等！），畢竟是別人把大致上的架構都寫好了，只要去從中找一些可以更改的地方改成自己喜歡的版本就可以了，相對來說也比較容易！

以上就是關於這個 blog 的歷史啦 ヾ (´︶`*)ﾉ♬
