# 引言

這學期的開始就給自己訂三個目標，也是上面藍色框框寫出來的

1️⃣ Tryna say YES and never regret

這主要是希望聽到當組長、負責人或所有有趣的事情都不要拒絕，因為知道自己對於這個職位會有很多的想法，既然有很多想法，不如就自己跳出來做，把想法拿出來，而不是在底下不知道該不該說；另外就是有什麼額外跳出來的活動都不要拒絕的去參與，雖然會把自己搞得很累，但之後想起來一定會是值得的，所以希望自己不管什麼事情都先試著說好，先答應再一步步做到最好，這樣就有更多故事可以譜出，成為一個豐富的人。

而這點，應該算是做的不錯吧！下面來看看這學期到底做得如何

2️⃣ Tryna be myself and do best

第二點是希望自己對於每一件事情都能做到最好，可能是學期初翻了一下高中的東西，難到當初交大數理資優研習的自傳和功課，想到讓我很感動的點，也想到上大學的我似乎很常對作業就只是交差了事，才希望這學期不管是什麼地方都能做到最好，最好的定義對我來說是要求都達到加上一些些自己的東西再加上額外學到的東西，不過這學期到後面似乎也是沒辦法了，甚至連作業都沒好好寫，唯一堅持住的只有客家社會探究

3️⃣ Be considerate

這邊是希望自己成為一個貼心的人，可以當別人身邊的小太陽，溫暖別人，其實就是從身邊相處的時候讓自己細心一點、敏感一點去注意一些小細節，觀察別人的需要，但這點我可以說我做的超級爛啊！很明顯是我做得到但我不願意做而已，到後半學期還是漸漸把自己封閉起來了。。

用一段開學第一天的日記總結這段引言

赫然發現，國中後的我是秉持著「一切都有最好的安排」，算是被動接受，既來之則安之的表現，但到高中經歷一些人事物，反而變成「人定勝天」，這大概是本質上的差異互相吸引，也是互相排斥的點吧！排斥不是不喜歡，而是兩種方法不一樣，但正是布一樣才會吸引到我，怎麼可以同時顧及兩個想法達到平衡，可能就是這學期的課題了

# 課程

上了大三選課都會失利了，這學期的班經、特教導論都沒選上，偏偏是我很有興趣的兩堂教程，就算不修完教程也想修這些課，不過如果真的上了這學期應該也是會完蛋吧！

# 資訊安全

相較於資安攻防，我覺得資訊安全稍微無聊了一點點，可能是因為聽前面密碼學還是有一點沒辦法接受（甚至覺得有開始摸到他的輪廓是在期中考當下），不過因為做大三專題的關係，對於密碼學這個領域相較其他人而言是起步比較早的，像是 security proof 的 game model、Diffie-Hellman 這種東西自己之前都碰過，相較下還是輕鬆一點，或許等離散的概念都起來之後，密碼學就會變得可以接受了吧？

澄清：所謂沒辦法接受不是不喜歡，而是覺得沒有到很懂，有一種「嗯我知道你在幹嘛，然後呢？」的感覺

至於專題，會選擇防火長城還是同一個原因，因為想了解防火牆，但對防火牆還是沒有很熟悉，所以在強迫自己接觸一次，看看有沒有什麼能再次了解的，絕對不是因為看中文文獻很開心

有一個可以分享的是在看一篇寫防火長城背景的文獻時，有提到中國加農砲 (The Great Canon) 的概念，就是說在防火長城上加裝砲火（加農砲），讓防火長城從被動防禦變成可以主動攻擊的角色，當下看完就覺得這個類比超帥！

但因為這學期作業比較不認真寫，算是少了兩次作業，讓資安變成這學期最低分，也是唯一一個沒有 A 的科目，可惡！

# 作業系統

作業系統是偽選修，明明是一門很重要的課

基本上就是圍繞在三個主題：virtualization、concurrency、persistent 上，說實話是不難，但作業可能因為 kernel code 在平常比較少碰到，真的要去改，如果沒有對整體有一定的認識，還是有一點難度的，加上 xv6 裝在 VM 的 QEMU 上，每次要跑都會讓電腦很燒，沒有督下去基本上也是跑不動，體感超差

嗯。。作業系統大概也是這學期唯一一科可以跟同學討論作業的課，雖然討論到後面好像變成都在等 Andy 老師，如果哪一天 Andy 老師都沒跟我們一起修課，我看我們應該都會死得很難 😦

# 大三專題：FHEwEC

延續上學期的研究，原以為這學期就單純每個禮拜開 overleaf 寫論文，然後實做出來就好，沒想到到最後幾乎是全部都翻過來重寫一次了

其實寒假後面在寫國科會計畫申請大概就知道我們這個研究發想出的 scheme 並沒有很好，限制太多了以至於泛用性超級差，所以會在國科會計畫申請交出去之後又全部重來發想幾個禮拜似乎是很正常的事情，也因為這樣，我們的研究時程整個拖延，後面寫論文和實作都是趕在 conference 截止以前才生出來，加上這學期我跟關平的時間又幾乎對不上，Neokent 也變很忙，每個禮拜的 meeting 只要一有意外就很難找時間補上，整體做專題的感覺是滿狼狽的

但回過頭來反思這次專題，會發現在去年剛開始的時候對於文獻探討的其實很不足，雖然 Neokent 給了 CPABE 的論文，並不代表只能看這一篇，甚至在知道做 FHE 的時候，對於 FHE 的整體歷史架構和原理應該都要有初步的了解，而不是覺得既然是做 FHE 之上的運算權限控管，就把 FHE 全部放掉，這樣在之後的研究影響其實滿大的，畢竟說不定有些 FHE 設計的 idea 就可以讓我的 solution 趁虛而入

最後恭喜一下，這份專題研究上了 113 年度國科會大專生研究計畫，也上了 2024 Asia Joint Conference of Information Security (2024 AsiaJCIS)，謝謝紀博文老師一年來的指導、王超老師對這個專題的題點與建議，還有我的好 partner 關平！

有一部分也是還不能接受為什麼套這個模型就能怎麼樣怎麼樣，這或許就是要實際花時間去了解 NN 這些東西在幹嘛才有解的地方

# 關於工作職責的一些想法

就這次活動而言我的身分是這樣的：對內第一組組長 兼 活動負責人 兼 手作課程講師

對內是第一組組長，在老師助教和組員之間溝通、分派工作、解決問題，需要的就是「負責」

為什麼當組長？基本上出於三個理由：一個是我很確定看別人當組長我會有很多想法，那既然有想法就自己出來當組長，就像軟工那學期每天的日記裡面不知道寫了多少 PM 哲學，明明自己不是 PM 也沒經驗，但就有一些覺得可以更好的地方，不過當時我沒辦法很帥的說「來吧！我來當 PM」就是了，如果當初說了全班大概會有 1/3 的人死得很難看，某種程度我也救了一群人（說什麼呢？）；第二個理由是學期初的目標吧！學期前兩個禮拜還是一直告訴自己對所有遇到事情都說好，強迫自己做一堆事情，這樣這學期結束就是很燦爛的，當初這個想法冒出來的時候也看到學期中在扁自己的樣子了，但我相信不管有沒有這麼做，這學期都能很順利的度過，既然都是很順利地結束，那就盡可能的多做一些事情，至少唯一能確定的是不這麼做我會後悔，所以衝動下就接了這個位置；最後的理由是喜歡資訊在我身上流來流去吧！是組員的時候如果需要一些決策但卡在上層，等待的過程其實滿煎熬的，但自己是上層的時候就不會有這個問題，某種程度上其實是回應到第一個理由。但我必須說，我當組長有很盡力避免第三個理由的問題出現，應該吧？

客家社會探究給了很多事情，一個禮拜跟你說要訂出主題，兩個禮拜要完成企劃書、海報、簡報，中間還有試作跟助教驗收等等，當你完成一件事情想要喘息一下就會收到通知跟你說「哈囉下一件事情來了」，最明顯是在企劃書、海報、簡報完成的時候想先處理期中考的東西結果算一算時間發現器材要先訂購、試作要先安排，結果想完這些事情就發現沒有休息時間了

在這麼多事情下，身為組長如果不先好好掌握所有資訊梳理完傳遞給組員知道，時程絕對是一拖再拖，所以說除了跳下去趕快把這些雜是解決也沒別的辦法

我也不知道我當組長適不適合，我也只是憑藉著高中科研的經驗來帶領一個團隊，不會扮黑臉，也不會社交，也沒有果決到所有事情可以十秒鐘內啪給出定論，更不是一個可以把錯綜複雜的事情一瞬間梳理清楚的人，說實話一路上我也很怕我的一個決定會讓組員開始討厭我，覺得這個組長沒什麼能力卻在那邊畫大餅，然後組就瓦解了，甚至有幾個禮拜二晚上會想東想西，想著學期初那個熱忱到期中又剩下多少，學期初我可以主動說我要當組長，學期中後會不會甚至看到組員都不敢講話了，事實證明這些狀況都有發生，一看到組員心裡的小劇場就開始一個一個兩倍速播放，而且是 enumerate 的方式把每種情況都走過

更疑惑的是不知道你們哪裡來的勇氣因為我第三個禮拜的一句「ㄟ我想試試看」就隨便我當上組長，總之謝謝你們的信任，也希望你們這幾個禮拜是玩的開心的！

對外是活動負責人，基本上跟報名學員的聯繫、mail 寄送都是我來處理

勝任這項工作是緊張的，因為知道報名學員年齡都偏大，年長者很大的問題是當我們說以 mail 作為主要溝通媒介時，一定會有很多人接不到訊息，很怕出現不該來的人出現在現場、抑或是說好會來結果當天消失的情況，但很幸運的是這群叔叔阿姨對於科技的使用比我想像中來的高，擔心的情況也沒發生

比較可惜的是當初想要試試看 gmail 有沒有群組的功能，我可以一次把該寄的信一次寄出，也能很有效率的統計所有人的狀況，但最後礙於期中考都沒去嘗試，變成每一封信都手動寄出，然後逐一核對回覆跟要求（身為資工系還做出這麼沒效率的事情。。）

最後是手作課程講師，也就是要能夠教會組員、學員靠自己的力量成功做出一個竹籃

說實話我自己根本不會編織，對竹籃也是一竅不通，但會接下這個位置大概只是對竹編有興趣，私心想趁這次機會玩玩看而已，而且當初討論主題的時候提出這個想法的罪魁禍首就是我本人，當然由我來承接這個位置

不過對於手作課程的準備用心程度，大概跟這學期上家教一樣而已，就是我知道要上什麼、知道每一步大概怎麼做，有稍微了解一些細節處理而已。但我也很慶幸我們組在事前有三次試做的機會，從第一次大家做出兩個超級歪又不美觀的竹籃、到第二次會注意一些小細節，實驗不同做法並熟練流程、到第三次抓住近乎全部該注意的事情，還有餘力紀錄發想屆時手作課程的上課方式，也做出一個超級完美的竹籃

# 不能忘記的事情：感謝

感謝從大二就想修的【客家社會探究】，在這學期真的選到，也沒出現撞課的情況，讓我能從這堂課中收穫很多

感謝這堂課的老師、助教以及客家文化園區，幫助我們處理很多行政上的事務，即便要我們自主辦一場活動，也省去了很多麻煩

還有謝謝每次都會關心我狀況的哲葦，雖然不是同一學期修的，但一知道我有修這門課就會不定時關心我，真的是小太陽！

最後有一群不能忘記的組員

記得我們在 DISC 人格測驗是一群由貓頭鷹跟無尾熊組成的小組，簡單來說就是偏向木訥但認真的一群人，最怕出現大家都不說話的機會超級高，但從籌備到現在，都沒有這個現象，至少訊息丟出去不到三分鐘就會有人讀訊息、給回應，處理事情的效率也很高，基本上工作一丟出去 deadline 前一定拿的到成品，而且都是認真做出來的，這樣的組合很輕鬆，似乎變成一位呼風喚雨的組長，但其實是底下的組員幫了大忙！

對我來說，文晶是我們這組最適合當組長的人，做事比較有魄力，也很有效率，重點是寫企劃書和成果報告書的時候都很有經驗，所以也很感謝文晶在這次活動中付出很多，對於活動的、對於組內的總總都是

懿芬是同為資工系的夥伴，但懿芬在活動當天的表現真的驚豔到我，我們活動現場有一個唯一一位弟弟，在整場都是叔叔阿姨的教室，大家都在做竹籃，那位弟弟不太可能有辦法乖乖跟著做（畢竟難度太高），看到這個情況，懿芬願意花大半時間坐在他旁邊陪著弟弟玩，讓弟弟對整場活動仍有參與感，這是我覺得很難能可貴的！

組上也有比較安靜的人，像是宇宸跟詠涵，總是安安靜靜把所有事情都處理妥當，像是活動當天的重點攝影師就是詠涵，幫所有人都拍出了專業的照片，偏偏我們其他人都沒注意到要記得幫攝影師拍照，真的是整場活動最可惜的地方。而宇宸雖然安靜，但在活動當天也是幫學員很多忙，名義上大家都是助教，但宇宸願意長時間陪一個學員，幫助學員從進度十分落後的情況一步步帶領到其他人所在的位置，這也是我很感動的地方，像我就會怕只顧其中幾位學員會讓學員覺得不公平，但正是因為組內還有其他人的協助，才能放心把所有小細節都交給組員，不用怕落單情況出現

世蕾雖然說前期可能比較忙沒出現，但在後期每次表現也都很亮眼，說實話他是很能帶領活動的人，從他對學員講話的口氣以及說出來的內容就能觀察到他對於帶活動應該很有一套，也是因為他，在活動當天學員比較能在需要的時候注意到我在台上說的話

最後想說的是佳翰，或許是整組裡面我聊最多的人，就我的觀察會覺得佳翰很用心，而且每次都笑笑的，總是會有很容易親近的感覺，記得佳翰有問過我一些服務學習的問題，希望經過這堂課有讓你真的體驗到服務學習課程的內涵，而且是玩得開心的！

記得我自己曾經說過，用心去對待的事情就可以寫下很多心得，看來這次也不例外呢！

# 關於課程的一些反思

這一段是節錄自期末個人作業中其中一題霹靂啪啦寫出來的東西。這學期有兩堂課的時間是老師親自在台上分享客家文化，讓大家更深入了解客家文化的面貌，但聽講的同時也反思了文化融合與文化衝突的事情

警語：這邊的言論純粹想分享自己的看法，我對各個文化的態度也是接受的，對我而言就是不同的東西、每個人有自己崇尚的方向，而對於不同人的不同想法，我也認為應該做到尊中與包容，另外也沒有覺得誰對誰錯，就像前幾句提到「每個人都有自己崇尚的方向」。

在課程中提過一個疑問:「為什麼閩南語就被稱作台語，似乎閩南語才是台灣的語言，而客語、原住民語等其他語言就不是？」確實我認為這個疑問有存在的必要，但或許可以花一些心力思考閩南語、客語、原住民語、國語甚至台語的意義是什麼，我想對於大多數說出這些話的人（此處指一般民眾而非提出者或特定利益團體及政府）來說，這只是一個稱呼來分辨不同語言，就像我的名字一樣只是辨別我和其他人不同身分的存在，在說這些稱呼的同時又有多少人真心認為台語就是台灣的語言，還是就純粹泛指閩南語而已？我的立場是如果這些稱呼帶有負面意義，那的確應該被捨棄或禁止，例如先前台灣會稱原住民為「番」，這種具負面意義的詞彙稱呼就應該捨棄，即便說出口的人也認為這只是一個稱謂，並沒有真的帶著負面含意，但因為該稱呼本身就具歧視意味，因此不應該被使用。當然，並不是要求閩南語以外的語言都要忍受不能成為所謂台語的資格，我同時也認為「台語」這個稱謂應該泛指所有在台灣這塊土地所使用的語言，沒錯，我的立場是站在地緣上，只要是在這塊土地上說出口、發展的語言，本質上都可以說是所謂的台語。

我想點名的觀念是：名字就像標籤一樣，當我們為了一個新事物命名時，就是給他貼上了一個官方的標籤，雖然這個標籤是讓我們認識他、分辨他，但也同時可能帶來價值觀上的誤差，能仰賴的還是一般民眾的理性思考，就像「駭客」這個詞彙其實是泛指透過資訊技巧解決問題的人，但我相信正在讀這篇文章的讀者剛剛第一眼的想法應該也是不好的那些網路犯罪攻擊者，其實這就是隨著時代演進、詞彙誤用導致的現況，但我們能說把這個詞彙拿掉嗎？似乎不行，在某些情況下反而會變成接受現況，並隨之轉型。

# 疾病、醫學與文明

這是為了「自然科學」領域選的通識課，不知道為什麼，以前排斥生物的我，現在要選這個領域的通識課，倒是都去找生物相關的

這門課每個禮拜都是一個講師來進行一小時的分享，包含各種醫藥領域或社會科學相關，接著授課老師來進行一小段總結，並帶大家進行反思，老師很棒的點是會帶大家嘗試使用 AI 科技，至少讓沒接觸過 ChatGPT 的人都接觸過，也在每個禮拜對他有多一點了解，說實話沒接觸 ChatGPT 的人還是很多，真的從這門課還是看到一堆人對於這些新科技、趨勢都不願意跨出去了解

# 啟發式演算法與解題應用

這是不在這學期一開始的規劃的課程，原先是以為會有進階程式設計，但宗哲老師說今年不開，改開這門啟發式演算法，才納入我的課表中

一開始的壓力真的很大，宗哲老師在第一堂課就說他打算把這門課開成「研究所入門」，意思就是在這門課給大家一些進研究所前的基本概念，比如說怎麼做研究，論文怎麼寫等等，至少真的進去碩班，不會一開始被指導教授電的很難看，或許是因為這一番話加上宗哲老師說每次作業都要用 IEEE 論文格式寫，所有有選這門課的大三生都跑去另一間的影像處理，最後留著的大三生就剩我一個人，要跟大四學長姐和一堆碩班的人修課壓力真的。。好大

壓力大不是沒理由的，宗哲老師的課其實不難，加上宗哲都會用白話文解釋每一個觀念，每個觀念間的銜接又講得很好，就課程內容來說完全沒有問題，是真的有每個禮拜待完三個小時就覺得腦袋裡多了很多真的搞懂得觀念，是認真學會的，這部分很好，但宗哲老師的課有時候會有分組活動，可能跟不認識的人、可能跟原本的組員，但基本上都是給你一個小時做出一個東西，然後用簡單的簡報報告一下，這之中問題就是一個小時的時間其實很緊湊，對我來說當時間一還給我，第一件在想的事情就是「完了我現在一點想法都沒有」，必須用很短的時間知道自己要做什麼，然後有系統的實作出來，再冷靜的分析，這是現階段的我還很努力學習的。反觀身旁的學長姊，很多真的都是在報告時講出很多小細節是我當下沒特別發現的，同時也很欽佩他們能在短短一個小時內看到這麼多東西

從這門課也看出一個小問題，這學期很多的實驗都發現很多人做實驗的步驟很奇怪，有用實驗結果去推假設，也有同時改了好幾個變因要去做比較，這些科學方法應該是國高中就會有的基本觀念，但不知道為什麼這學期還是看到好多這種問題

很開心也很感謝的是在這一學期有很和善的學長學姊跟我同組做作業和平常討論，是真的都在一個很有溫度的情境下做這些作業，完全沒有壓力，這點滿難得的！

# 職業教育訓練暨生涯規劃

是這學期唯一的教程課，因為是教程的必修、又是淑禎老師開的課，加上只有半學期又不密集的一學分密集課程，就安排在這學期了，說實話 12:10 啟發式下課又接 13:20 的這邊，加上啟發式下課之後又常常會留下來討論作業，根本沒辦法好好吃午餐和休息真的很累。。

這門課的內容比較偏向「技職教育」，八個禮拜的時間內大概有一半時間在說明技職體系，畢竟師大的教程出去也是可以用在高職，而我們這些（I mean 大多數）普高上來的學生，對技職體系其實很不清楚，這門課的存在就有很大的必要性，不過個人是偏好課程後半在談生涯規劃的部分

這門課的分組認識了一位數學系、一位音樂系、一位美術系和一位化學系，很剛好都是男生，說實話這個組合其實很酷，課程前兩個禮拜我甚至拿這門課的分組和輔原的時候比較，當初遇到兩個美術系學長真的都是天使，也是這樣超級喜歡教程，但一開始在這次的組別卻有那麼一點點沒辦法融入，每個人講話都很簡短，對教育的理念、思維跟我落差也滿大的，算是讓我反思到這一路以來教程上很順利，或許都是因為沒遇到教育理念差太多的人，但這個情況一定會發生，尤其當一個人的想法跟我大相逕庭的時候，我要用什麼角度、心態去面對，就變得很重要，慶幸的是學期初的我還很有動力 social，從每次討論和接觸反思彼此教育理念的差異，才漸漸接受這些組員，到期末甚至覺得他們也很讚

弘意老師本身是師大工教系的學長，也是前幾年師鐸獎、十大傑出青年得主，聽他的分享真的能感受出他對學生、對教育的熱忱。BTW 弘意老師也是今年不分區立委民眾黨候選人，雖然沒被選上，但也是有一個參政的經驗！

這邊想要分享在演講最後弘毅老師提出來的一個點

弘意老師推薦「多帶社團、專題」，因為帶這些額外的活動，才是真的有機會讓身為老師的我們去動腦思考，當一個老師停止思考，覺得不需要備課就可以上課時，身為老師的熱忱、動機和優勢也就不見了

這段話對我來說是一劑很強的強心針，撇除在資工系的環境，在這之外要能有這種思維的人真的算少，我也不敢說我自己就不是，但這算是這幾年我奉行的原則了吧！也算是努力在打自己變成這個樣子

另外想分享的是演講後跑去教室前面找弘意老師和淑禎老師 QA 的紀錄

1️⃣ 國中技藝班是給不會讀書的孩子去的，關於這個現狀老師的看法如何，而要如何改善

這是一直以來的問題，也講好多年了，但卻沒有變化。這部分是要國中端、政府端和家長端都要同時改變想法才會有的結果。因為大多數家庭環境還是覺得最好就是先讀書，讀書比較有前途

這邊新增一點握自己的想法：技藝班是給不會讀書的人去這個想法就不對了，畢竟成績好的同學如果有接觸技職體系的必要與能力，也應該讓他去試試看，而不是讓給其他學生，究竟多元入學是要讓所有人可以採用更多元的方式入學，還是提供備案給成績差一些些的學生使用，這是要好好思考的議題

2️⃣ 技職體系沒有生物科，那對對生物有興趣但成績不足的孩子應該如何應對

還真的沒有生物相關的類、群、科，說能不能多開一個群，應該也是不太可能，畢竟現在少子化，所以只能期待那個孩子可以努力把自己的成績提升，想辦法擠進高中學習

3️⃣ 科技業不當老師走業界發展至少是老師的兩三倍，但對於教育還是有興趣、熱忱，老師覺得應該如何規劃

畢業後先把教檢拿到、再去業界工作，最後有機會再回學校。學校如果走技職體系是絕對沒問題的，畢竟技職缺老師，但業界如果畢業後一次晚個十年再回到業界基本上也沒什麼機會了。當然學校不論老師或學生，都喜歡年輕老師，這是無可避免的事情，但以職位來說是不用擔心的！

# 生涯規劃 & 生涯願景

老師：請跟隔壁同學說什麼是生涯規劃
我：規劃生涯？
化學系同學：？？？

後半段的課程著重在生涯規畫方面，其中有一堂課是討論生涯願景，雖然說是帶我們做這個活動，目的應該是讓我們以後有方向帶領學生做這些生涯探索，但教程很不一樣的就是即便宗旨在以後的教育，這些課程其實現階段都是在一次幫助我認識自己，課程中的反思真的很難能可貴，所以決定分享一下這堂課的過程！

一開始老師發給大家一張長條紙，說請大家折成 10 等分，假設一等分是 10 年，你最多有 100 年的壽命，你希望活多久，如果比 100 還要小，就把多出來的撕掉，然後假設以 10 為單位，也不能去搶別人不要的壽命

當時的我想說，人生有很多事情要做，有 100 當然要活 100 歲，所以沒特別裁，回想起來超喜歡這個想法的，以前的我或許會覺得「啊 80、90 就不錯啊！」，但現在的我真的有改變

接著是要在 0 - 20 歲（也就是前兩格）寫下至今最滿意的三件事，這部分是歷史，活了 20 多歲，一定有一些覺得很自豪的事情，短短 20 年間，我們到底做了些什麼？

我的部分。。第一想法還真的想到高二那邊，但還是很隱晦的寫一個「真正認識自己」，另外兩個其中一個是想到最近壢中的計畫還有 PASSION 的實習，讓我變成「有專業能力貢獻在社會或身邊」的人，另一個是影響我很多的科研經歷。當然還有更多，小時候的打擊、家庭給予的環境、以前每個禮拜都會出去玩、跟爸爸媽媽關係仍然很好等等，這些都是值得我驕傲的點

寫完之後就是跟大家交流，從這邊就可以反思自己寫的事情和別人寫的事情差別是什麼？也能更了解自己的組員對於他自己在意的點是什麼？當然這不是比較，而是透過其他人的眼睛去看到更不一樣的世界，again，反思跟交流是很重要的環節

再來是把這 20 年的歲月撕掉，撕掉收起來是因為這些都是歷史，沒辦法再改變了，而還有機會改變的就是仍然空白的那些格子，因此下一步是在這之中寫出接下來人生一定要達成的三個願望

這時候我就是 greedy 的人了，一共寫了五個，但每一個都是認真的，當時寫也很清楚十年之後再看絕對會不一樣，但這次的活動會完完全全反映出 21 歲的我在想什麼

1️⃣ 是想要環遊世界、深度旅遊，這是小時候的夢想，國中的願景（aka 導遊），雖然現在不說了，但還是在我心中慢慢茁壯，也是我認真想達成的夢

2️⃣ 是懷有熱忱、持續服務，這包含了教育上、偏鄉上、做志工、特教等等，或許還會慢慢補上一個以前不是很喜歡的長青

3️⃣ 是擁有一間工作室，有一個安身立命的避風港，這間工作室可以小，但工作室等於我家，最好還可以結合咖啡書店的想法，順便服務周遭學童，這個是我真的想過的

4️⃣ 是擁有自己的家庭，嗯。。淺顯易懂

5️⃣ 是希望自己成為有影響力的人，有一個代表性的貢獻，在這社會上被看見

最後就是再跟同學交流一次，而我滿喜歡這組同學給的回饋：每天跟自己說三句鼓勵的話、重視自己的每一個付出就能持續做下去，又是些我都懂的道理，但有人點出來感覺很不一樣，會有一點點不一樣的思維跑出來再反思一次

# 家教

家教的部分，感覺上一上會有一些感覺飄掉，比如說連上了兩個禮拜突然發現最近上家教好像變成 routine 一樣，沒有特別對教育的感覺，就知道應該花時間反思一下最近的教學成效，在這種情況下教出來的東西肯定不會多好

⚠️ 以上，真的不要亂學，每次上家教都會蹦出這些有的沒的事情，但都不是什麼好事，要當老師還是要好好備課的！

再加上我家弟弟要升高三了，他要升學我也要升學，真的有一種自顧不暇的感覺，但不可能把他放掉，這樣太不負責任了

# CPE

CPE 從去年五月去北教大開始用 C 寫兩題，接著又去台大用 C 還是寫兩題，到今年因為抱不到這兩間，所以跑去政大，加上刷 leetcode 的習慣，慢慢用 Python 刷到四題甚至五題，有在刷 leetcode 還是有點差別的，接著就是希望大四畢業前能進階變成六或七！

# CLHS DOOS

這是 1/13 晚上不知道發什麼神經，突然決定要做的事情，甚至為了這件事情一個晚上打了 80% 的企劃書

這是壢中熱食部線上訂餐系統計畫，主要是因為自己知道沒什麼經歷，過了軟工又覺得對於前後端都沒有很了解，想在一次機會去實作一個東西，剛好高中的時候常常聽大家說要做這個卻都沒做到，就決定來搞看看，也因此拉了一些人陪我，一開始是想到這個企劃，忍不住就跟瑋宸說，想找他一起當 PM 把這個做大，甚至 1/19 還回壢中找龍哥討論，接著又陸陸續續找白白跟前 PM 可伊一起玩，原本是想弄得更大，要跟高中生一起玩，但後面這部分沒實現，一部分是高中生似乎沒興趣，一部分是我這邊也無法負荷

說實話這個計畫是私心起來的，但我是認真想把它弄好，從需求規格書、設計規格書那邊就想要每一步都做好，但真正開始才發現有好多問題，一次大量的東西下來，會不知道從何下手，也沒辦法有系統冷靜分析整理這一切，加上後面事情又越來越多，原本每週四晚上都會上線聊天討論的，到後面也直接取消

以這學期來說是可惜的，但還是會找時間弄好吧！當然這陣子也有在想是不是自己當初又想的太美好了，每次在進到專業知識的 part 之前都能處理得還不錯，但真的要開始動到專業知識時就會開始拖延，好大的問題

# 腳踏車

這算是為了大四做準備，說要去腳踏車環島作為我的大學畢旅是認真的，但很明顯以我目前的體力跟技術完全不行，所以想說這學期讓自己多期一點教踏車，就在開學那一陣子騎了兩次，一次是跟益鳴去環騎台北，環騎台北大概有 60 km，會透過各個河濱步道環繞台北是一圈，但因為我直接說是是看挑戰騎，就多了木柵那段山路，或許是因為那陣子下雨的關係，地面還是潮濕的，我為了閃後面的車就決定往山壁靠一點，結果在下坡路段騎到水溝蓋上，水溝蓋又很滑，直接摔車翻了兩圈，是沒出什麼事，但說實話現在對於下坡路段多少有點陰影，也很可惜那次的環騎台北就此泡湯（也沒下次了）

之後是趁 228 連假因為政府說要開始實施前 30 分鐘免費，加上又沒事，就決定再騎一次，從學校騎到社子島，主要是知道自己如果不騎，真的會對腳踏車產生陰影，之後要突破又很困難，所以強迫自己去接觸。結果不知道為什麼一到大稻埕，看看距離，就又往淡水那邊騎，第二次騎跟去年九月騎的感覺不一樣，比較有方向感，也沒有迷路（好啦！有迷路 20 分鐘），但時間卻花得差不多，主要應該是之前先騎大馬路到北車那一帶再切進河濱，這次是全河濱的挑戰，距離也相對較遠，差不多 30 km 多

# 展望

升大四其實展望也很簡單，所有事情都好好的就好了吧！接下來一年的叡揚實習就好好跟公司進度，盡量學我需要的技能起來；升學跟學業的部分，就等我找到答案再說，感覺這也是可以寫一篇的事情！

大四，又是新的通勤人生，應該會很累吧！但希望維持自己的初衷，再去追一次夢！
